para rodar o index.js no cmd: node index.js

para rodar os testes no jest uma vez: npm run test
para rodar os testes no jest automaticamente após alteerações: npm run test:watch
para rodar os testes e gerar relatório: npm run:coverage

sobre a nomenclatura e estruturação de pastas dos arquivos de teste:
https://cursos.alura.com.br/course/nodejs-testes-unitarios-integracao/task/110307
